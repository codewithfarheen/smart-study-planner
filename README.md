# 📚 Smart Study Planner & Task Manager

A modern web-based task management application with multi-language support and clean UI.

## 🚀 Features
- Add tasks with date and time
- Mark tasks as completed or pending
- Filter tasks (All / Completed / Pending)
- Progress tracking system
- Multi-language UI (English, Hindi, French, Spanish)
- Task translation using Flask API
- Interactive UI with particles and animations
- Local storage support (data saved in browser)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Python (Flask)
- REST API

## ▶️ How to Run

### Run Backend
pip install flask flask-cors requests  
python app.py  

### Run Frontend
Open index.html in browser

## 💡 Purpose
This project demonstrates frontend + backend integration, API usage, and UI design skills.
